Introduction

Dependency-free

Litebox doesn't depend on any third-party scripts like jQuery or lodash. Include one script and you're good to go.

Lightweight

Compressed and minified, the script only weights around 10 KB, which makes it load quickly into your website.

Customizable

You can customize the behaviour, such as the animations, selectors or translations. It adapts to your needs.

Modern

Litebox is built using the latest version of EcmaScript and offers quiet readable and beautiful code.

Cross-browser support

Being compiled into ES5 using Babel, Litebox is backwards-compatible to Internet Explorer 10 and other browsers.

Sexy

Although including a simple, yet nice looking default theme, you can customize any CSS to your needs and CI.

Examples

Getting started

  1. Download the latest version of Litebox here and extract the ZIP file on your hard disk.
  2. Copy the following files into your project folder:
    litebox.min.js, litebox.min.css, icons.svg
  3. Include the CSS and JavaScript into your website:
    • Place the CSS into the head of your HTML:
      <link rel="stylesheet" href="/path/to/litebox.min.css">
    • Place the JavaScript before the </body> tag in your HTML:
      <script src="/path/to/litebox.min.css"></script>
  4. Initialize Litebox by pasting this code into a different script file:
                    
                  var box = new Litebox();

                  box.init();

Options

Option Type Default Description
animations Boolean true Enables or disables animations.
gallery Boolean true If set to true, Litebox will scroll through all images on the page
